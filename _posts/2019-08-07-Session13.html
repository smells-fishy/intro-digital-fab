This page is a compilation of the multitude of ways that I have managed to break Autodesk's widely acclaimed Fusion 360. Note that not all of these are particularly well documented, due to issues with replication.

Disappearing bodies

Disappearing bodies are probably the scariest of all of these bugs. Fortunately, it generally occurs under very specific circumstances: when you save a file and then quit Fusion while it is updating. In order to fix this issue, you just need to restore a previous version of your file and start working again. Be sure to not save and exit until the update is done downloading.

Minimize

Floating Browser

Sometimes, it is possible to get the brower to stay on the screen even after Fusion is minimized. This bug seems to be relatively random, but occures most consistently when you manually minimize the window right after alt-tabbing (or cmd-tabbing for Mac) out of the window and then reopening. Occasionally, this will occur when you alt-tab, which is annoying because it gets in the way. If you click on the browser, Fusion will show up once again.

Minimize

Names get cut off

I just chanced upon this one after reaching 100 bodies in a single file. Apparently, Fusion isn't smart enough to add the extra character space so it ends up... taking? I'm not quite sure how this one works but those are the circumstances under which I encountered this issue. Solution: resize the browser.

Disappearing constraints

This one is frustrating because you actually need over 1000 apm to fix it in a reasonable period of time. This bug honestly just happened one day and made me very displeased.
